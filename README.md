# C-operation-priority
C语言运算优先级快速记忆法
关键字 ： C语言运算优先级快速记忆法    C语言运算优先级口诀

运算优先级口诀（读法）：
成单 陈佳怡 等卫子夫

口诀记忆法：
成单  陈佳怡美女  古代美女卫子夫


运算优先级口诀（写法）：
成单 乘加移  等位辑赋

口诀说明如下（序号越小，优先级越高）：
序号	  口诀	  解释	        运算符 	
1   	  成	    成员	        .   ->	
2	      单	    单目	        ++   --   *   &  !  ~    sizeof	
3	      乘	    乘除	        *   /   %	
4	      加	    加减	        +  -	
5	      移	    移位	        <<  >>	
6	      等	    大小不等	    ==  !=    <=  >=     <  >	
7	      位	    位与或	        &   ^   |	
8	      辑	    逻辑与或	    &&    ||	
9	      赋	    赋值	       =  *=  /=  +=  -=  &=  |=  ^=
